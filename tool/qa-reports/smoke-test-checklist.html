Smoke Test Checklist · v1.1

Agaram — Does the APK work?

Full-app regression pass: ~55 checks, ~60 minutes across three testers (one Admin, one Member, one President).

How to use: Install the APK on a phone, then work down the list. Tick the box if the step does what the "Expected" column says. If it doesn't, circle the ID and write what happened in the margin.

Best run: 3 testers in parallel — one President, one Admin, one plain Member — so every role's flows get tested at once. Solo runs work too — create 3 test accounts and switch.

Prerequisites: valid assets/fcm-service-account.json in the APK (not the placeholder) for pushes; admin account authorized with Google Drive if you're running the archive checks (ARCH-*).
